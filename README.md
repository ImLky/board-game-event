# 부루마블 이벤트 웹 게임

## 프로젝트 소개

팝업스토어 방문 고객을 위한 이벤트용 부루마블 웹 보드게임입니다.
로그인 없이 URL 접속만으로 즉시 참여할 수 있으며, 주사위를 굴려 다양한 경품을 획득하는 방식입니다.

> 본 프로젝트는 **포트폴리오 목적으로 구현**되었으며, 실제 팝업스토어 이벤트와는 관련이 없습니다.
> 등장하는 쿠폰, 경품 등은 모두 가상의 정보입니다.

## 주요 기능

- 1~4명 멀티플레이어 지원 (턴제)
- 6x6 CSS Grid 기반 보드판 (외곽 20칸)
- 주사위 굴리기 애니메이션
- 말 순차 이동 애니메이션
- 칸별 이벤트 (쿠폰, 미션, 꽝, 보너스, 특별경품)
- 보너스 칸 도착 시 주사위 한 번 더
- 플레이어별 게임 결과 요약
- PC / Mobile 반응형
